# Lucy Rainbow Daycare 🌈

Official website for Lucy Rainbow Daycare — a licensed, family-based daycare serving Richmond Hill, Queens for over 20 years.

- Fully licensed family group daycare
- DOE-certified 2K & 3K programs
- Ongoing training with Queens Community House
- After-school pickup & transportation
- 📞 (718) 850-3210

## Running the site

Open `index.html` in any browser — no build step needed. Single-file HTML/CSS/JS with photos in `images/`.

To publish free with GitHub Pages: repo Settings → Pages → Source: `main` branch → root.
